# Planning-poker

image:https://img.shields.io/badge/vert.x-3.9.0-purple.svg[link="https://vertx.io"]

Simple planning poker written in Kotlin with Vert.x.

## Building

To launch tests:
```
./gradlew clean test
```

To package:
```
./gradlew clean assemble
```

To run:
```
./gradlew clean run
```
